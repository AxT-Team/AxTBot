# AxTBot-Public
基于QQ官方PythonSDK开发的AxTBot-Public机器人程序

## 相关注意事项:
Hypixel查询是基于Spelako项目修改而来，此存库只提供用于启动PHP服务器API接口的代码，其他代码请到原存库进行获取<br>
https://github.com/Spelako<br>
使用方法:把SpelakoCore文件和index.php放在同目录下，使用start.bat启动即可（Linux自行编写启动脚本）
